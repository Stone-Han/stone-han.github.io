如何使用ppt制作高清海报(会议论文poster)

发表于 2018-09-03 分类于 教程

最近参加会议需要制作poster，在ppt上选好模板做好之后，发现实验图不清晰。下面记录一下生成高清海报的过程：

  • 首先实验图要清晰的。先把实验图从pdf中转成jpg，在转换的时候有个设置，可以全部设置成最高级别。生成的图就很清晰了。

  • 然后插入ppt的时候尽量不要有压缩。这个可以在ppt的选项-高级中设置，插入的图片

  • 最后ppt生成的图片也要是清晰的，这里需要用到注册表，win+R 输入regedit，找到ppt对应的键值。新建一个ExportBitmapResolution，数值为十进制的300.然后把ppt转成jpg就可以啦。

金融思维7：资本资产定价 股票/公司估值

发表于 2018-05-08 更新于 2019-01-19 分类于 金融

日常生活中的定价：实物或者服务是按照成本+利润。但是利润对于不同市场是不同的。对于白菜市场，由于竞争比较充分，财务成本弥补了机会成本，所以经济利润为0。所以定价为成本+机会成本。对于竞争不充分的，可能还包括一些溢价。

同样，对于资本资产定价来说，有两个思路，一个是成本，一个是机会成本。
